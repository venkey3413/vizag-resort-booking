# Vizag Resort Booking - Flutter App

A Flutter mobile application for the Vizag Resort Booking platform.

## Setup

1. Install Flutter SDK
2. Run `flutter pub get`
3. Add your images to `assets/images/`
4. Run `flutter run`

## Features

- Home screen with carousel
- Service tiles for different booking options
- Responsive design
- Material Design UI
